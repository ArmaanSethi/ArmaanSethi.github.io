# armaansethi.github.io

Personal website - a curated collection of things I find interesting.

## How to Update

Edit `_data/links.yml` to add new links:

```yaml
- title: "Paper or Article Title"
  url: "https://..."
  year: 2024
  note: "Brief thoughts on why it's interesting"
  favorite: true  # Optional - adds a star
```

Push to GitHub and the site auto-rebuilds.

## Structure

- `_data/links.yml` - All the links (edit this!)
- `_config.yml` - Site settings, social links
- `index.html` - Template
- `style.css` - Styling
